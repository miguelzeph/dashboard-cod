Regular Expression Suggestions
Regular Expression tips: Follow below some useful examples how to find codenames with Regular Expression on the search panel:
  • AZD If there isn't codename just with "AZD", our search bar will returning everything that have AZD in the name, like:
  • AZD.* This regular expression matches any string of characters that contains "AZD" followed by zero or more of any other characters. For example, this would match "AZD123", "AZD testing", "AZD123!", "123AZD", and so on
  • ^AZD.* This regular expression matches any string of characters that starts with "AZD", followed by zero or more of any other characters. The "^" symbol at the beginning of the expression means that it must appear at the beginning of the string. For example, this would match "AZD testing", "AZD123!", but would not match "123AZD".
  • ^AZD.*5$ This regular expression matches any string of characters that starts with "AZD", followed by zero or more of any other characters, and ends with the number 5. The "$" symbol at the end of the expression means that it must appear at the end of the string. For example, this would match "AZD1235", but would not match "AZD5" or "AZD123".
  • ^AZD(-| )[0-9]+ (for space you can use a simple space or \s) This regular expression matches any string of characters that starts with "AZD", followed by either a hyphen (-) or whitespace (\s), followed by one or more digits. The parentheses around the hyphen and whitespace indicate that it can match either one. For example, this would match "AZD-123", "AZD 123", but would not match "AZD123" or "AZD--123".
